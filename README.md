# hypercube

A contemplative, zero-dependency visualization of the n-dimensional hypercube.

_Work in progress — see `docs/mathematics.md` for the underlying mathematics._

## Run

```sh
python -m http.server   # ES modules don't load over file://
```

Then open <http://localhost:8000>.

## Test

```sh
npm test                # node --test, pure math core only
```
